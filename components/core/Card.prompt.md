Generic elevated surface — warm cream card with a soft plum-tinted shadow, the base for any grouped content block.

```jsx
<Card elevation="md" padding="var(--space-6)">
  <SectionHeading>A card, breathing</SectionHeading>
</Card>
```

Elevation: `sm` / `md` (default) / `lg` map to the shadow scale. Never use pure black shadows — the tokens are already plum-tinted.
