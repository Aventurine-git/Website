Pill-shaped call-to-action button in three variants, used for the site's few soft invitations (never hard-sell CTAs).

```jsx
<Button variant="primary">wander in</Button>
<Button variant="secondary" size="lg">find me</Button>
<Button variant="ghost" size="sm">learn more</Button>
```

Variants:
- `primary` — solid aventurine green fill, dark ink text. Use sparingly, for the one action that matters most.
- `secondary` — opal-trimmed pill on card surface. The default for most actions.
- `ghost` — outline only, for tertiary actions.

Sizes: `sm` / `md` (default) / `lg`. Set `disabled` to fade to 50% opacity and disable pointer events.
